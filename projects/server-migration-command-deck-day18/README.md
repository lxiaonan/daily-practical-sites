A local-only server migration command deck that turns source/target server inputs into risk scoring, a six-phase migration checklist, generated Bash commands, rollback guidance, Markdown/Bash exports, local progress persistence, and a meaningful three.js migration topology.

- Annoying step removed: scattered migration notes become one auditable plan with commands and rollback checks.
- Without it: users manually piece together blog posts, shell history, and risky ad hoc checklists.
- Non-technical workflow: fill hosts, choose components, review risks, copy/export plan, execute manually after verification.
