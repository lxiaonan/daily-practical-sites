## Notes

- This project puts the Windows scanner, browser review UI, risk explanations, size-ranked report, and DryRun cleanup script generator in one repository.
- It was chosen from same-day demand around C drive red-zone cleanup, unclear junk-file risk, and distrust of paid or fake cleaner tools.
- The shipped app does not directly delete files from the browser. It uses local PowerShell scanning, JSON import, and DryRun-first script export.
- Updated for non-technical users with `START-HERE.bat`, `SCAN-C-DRIVE.bat`, and a release zip package.
