## Notes

- This project turns messy download folders into a safe rename rehearsal workflow with rule composition, conflict checks, local snapshots, CSV export, and PowerShell export.
- It was chosen from same-day research around batch file renaming, media compression, local file organization, and asset cleanup needs on Linux.do and Juejin.
- The shipped static build is honest about browser limits: it does not directly rename local disk files, and instead exports a reviewable PowerShell script with `-WhatIf` enabled by default.
