## Notes

- This project turns screenshots into a searchable local index with OCR, reviewable transcripts, tags, status filters, and JSON/CSV/Markdown export.
- It was chosen from same-day demand around screenshot OCR, batch recognition, local indexing, and searchable screenshot archives.
- OCR uses Tesseract.js in the browser. If model loading fails, users can manually paste transcript text and still use search, tags, and exports.
